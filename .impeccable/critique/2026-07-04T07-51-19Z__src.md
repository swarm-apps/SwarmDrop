## 反模式判定

**LLM 评审**：不会被一眼认出是 AI 做的。卡片最深两层无套娃、全局单一金色强调（语义状态色除外）、glass 严格限于结构容器、无渐变 hero 无同质卡片网格。配对码签名组件、Mono Truth 纪律、双半径体系是"有明确观点的人"才有的细节。唯一擦边：主页"说明式 hero + 三统计磁贴"（见 P2）。

**确定性扫描**：detect.mjs 全量扫 src，仅 2 条 advisory，且核对后均为假阳性——`src/index.css:54` 与 `:105` 的 `--glass-accent-bg` 渐变色标（暖奶白 / 暗化金，色相 78 正是 Hive Gold）未登记进 DESIGN.md 与 `.impeccable/design.json` sidecar，属文档登记缺口而非视觉漂移。零真实反模式。

**浏览器叠加层**：目标是 Tauri 原生 webview 单窗口，无浏览器 tab 可注入且唯一窗口被走查占用，已按规跳过；以 CLI 扫描 + 实机截图走查替代。

## 总体印象

一个真正被认真设计过的工具界面，站得住"Linear / Raycast / 1Password 用户会不会信任它"的检验。强弱高度集中：状态诚实、美学极简、一致性三项真实卓越（各 4 分）；错误恢复（1 分）、高效操作（2 分）、错误预防（2 分）是三处真实欠缺。最大单点机会：把传输失败从"死胡同"变成"可重试的低谷"。

## 做得好的

1. **配对码签名组件**（`/pairing`）：每位数字独立 glass chip + mono text-3xl + 过期倒计时 + 右侧安全安抚文案（"只建立你确认过的连接"）。把最高信任时刻用最冷静的方式呈现。
2. **Mono Truth 全局纪律**：peer id、会话 UUID、multiaddr、速率、配对码、MCP JSON 处处等宽 tabular 无例外。对"可信工作台"气质的贡献超过任何配色。
3. **MCP 设置一等公民化**：状态/端口/自启开关/可复制 JSON 配置（点名 Claude Desktop、Cursor），完美落实"AI 是基础设施不是噱头"。

## 优先问题

**[P1] 失败传输详情是死胡同（错误恢复）**
- What：失败详情只有红叉 +"不可恢复失败"（且顶部徽章与中心区重复出现），无人话原因、无"重新发送"、无下一步。
- Why：违反 Nielsen #9，把用户永久留在情感低谷；71 个文件的发送失败后要手动重选全部文件代价高昂。
- Fix：给出人话原因（对方离线/连接中断/对方拒绝）；主操作"重新发送"（同载荷+设备重开发送流）；删掉重复标签。
- 建议命令：/impeccable harden

**[P2] 主页顶部 hero + 统计磁贴对老用户是冗余 chrome（认知负载）**
- What：常驻说明式 hero（"发现设备，配对，然后发送文件"）+ 三统计磁贴，计数与下方列表重复，占首屏 ~160px。
- Why：擦边 PRODUCT.md 的"通用 SaaS 仪表盘"反面参考，破坏其余页面守得很好的单一焦点。
- Fix：已配置用户收成紧凑标题（或仅 0 配对设备时显示引导）；磁贴并入面板标题。
- 建议命令：/impeccable distill

**[P2] 缺高效用户加速器（灵活性与效率）**
- What：零快捷键、无 ⌘K 命令面板、列表无多选/批量。
- Why：目标受众（Raycast/Linear 用户）默认键盘优先；删 11 条记录要点 11 次。
- Fix：⌘K 命令面板或至少 ⌘, / Esc / ⌘F；列表多选 + 批量删除/归档。
- 建议命令：/impeccable delight（快捷键层）+ /impeccable shape（批量操作）

**[P2] 破坏性删除无确认/撤销（错误预防）**
- What：每行 trash + 面板级"清空活动记录"直接执行。
- Why：一次误点即不可逆删除记录。
- Fix：确认对话框或撤销 toast；逐行 trash 收进 hover 显示。
- 建议命令：/impeccable harden

**[P3] 顶栏纯图标导航 + share-target 空载荷态语义矛盾**
- What：收件箱/传输/设置纯图标依赖记忆；share-target 空载荷时标题"准备中"与正文"文件已全部移除"矛盾且 CTA 仍启用。
- Fix：图标补文字或 hover 展开；空载荷态切换标题与主操作。
- 建议命令：/impeccable clarify

## 次要观察

- "局域网 0ms"像 bug，建议"局域网 · 直连"。
- 设置页 NAT 状态常驻"未知"，反损"状态诚实"强项。
- 主页配对码常驻暴露 + 持续倒计时再生，既是常驻动效也是常驻秘密。
- 传输详情可补平均速率（mono）完善遥测感。
- 文档卫生：把 `--glass-accent-bg` 两组色标补登进 DESIGN.md 与 `.impeccable/design.json`，让漂移检测保持零噪声。
