description: SwarmDrop 已完成与计划中的能力。
---

SwarmDrop 已经能稳定地端到端加密传文件，仍在持续打磨。

## 已完成

- [x] **P2P 网络**：libp2p、mDNS、Kademlia DHT、Relay、DCUtR。
- [x] **设备配对**：6 位配对码、局域网直连、生物识别解锁。
- [x] **文件传输**：端到端加密、实时进度、历史记录。
- [x] **多语言**：简体中文、繁体中文、English。

## 进行中

- [ ] **断点续传**：传输中断后从 checkpoint 继续，不必从头再来。
- [ ] **移动端打磨**：Android 端体验与桌面对齐。

## 计划中

- [ ] **MCP 集成**：让 AI 助手直接通过 SwarmDrop 发文件。
- [ ] **iOS 支持**：把全平台版图补全。

路线会随社区反馈调整。有想法欢迎到 [GitHub Issues](https://github.com/swarm-apps/SwarmDrop/issues) 提。
