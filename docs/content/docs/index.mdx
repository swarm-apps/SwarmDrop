SwarmDrop 是一款**去中心化、跨网络、端到端加密**的文件传输工具，定位为「跨网络版的 LocalSend」。
无需账号、无需中央服务器，文件直接在设备间点对点传输，且**只有收发双方能解密**。

## 它解决什么问题

把文件从一台设备传到另一台，今天的选择都不太理想：

- 网盘 / 即时通讯：文件经过第三方服务器，隐私无保障，还常被压缩限速。
- LocalSend：体验很好，但只能在同一局域网。
- Send Anywhere 一类：能跨网络，却要走中转服务器。

SwarmDrop 把 LocalSend 的「装好即用、点对点」体验，扩展到**任意网络环境**。

## 核心特性

- **跨网络自动选路**：局域网 mDNS 直连、DHT + DCUtR 打洞、中继兜底，自动选最快路径。
- **端到端加密**：每次传输独立生成 256-bit 密钥，XChaCha20-Poly1305 加密，中继也看不到明文。
- **零配置**：不要账号，不要中央服务器，可选自建引导节点。
- **全平台**：Windows、macOS、Linux 桌面端，Android 移动端，共用同一套 Rust 核心。
- **开源可自托管**：MIT 协议，引导节点可自建。

## 与同类工具对比

| 维度 | LocalSend | Send Anywhere | **SwarmDrop** |
| --- | --- | --- | --- |
| 网络范围 | 仅局域网 | 跨网络（中转） | **跨网络（P2P）** |
| 服务器依赖 | 无 | 有中转 | **无**（可自建引导节点） |
| 端到端加密 | 本地传输 | 可能经服务器 | **✓** |
| 开源 / 自托管 | ✓ / 不需要 | 否 / 不支持 | **✓ / ✓** |
